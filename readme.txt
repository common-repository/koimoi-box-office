== Description ==

= What does KoiMoi Box Office do? =

Keep your users upto date with the latest box office numbers. KoiMoi Box Office Widget takes live data from koimoi.com about the top movies on the Indian Box Office and displays the top 5 as a widget on your blog!

It has
1. Live data
2. Slick design
3. Color customization

For latest Bollywood news, box office updates, movie reviews, photos, videos, and more, visit [koimoi.com](http://koimoi.com/ "koimoi.com")!
